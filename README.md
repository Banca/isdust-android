已实现功能:   
              1.校园卡  
                    校园卡余额，消费纪录查询，校园卡修改密码以及挂失  
              2.上网  
                    快通有线，cmcc、ChinaUnicom一键登录，剩余流量/时间查询  
              3.课表  
                    学生个人课表按周查询，空自习室查询  
              4.外网访问  
                    在外部网络能进行信息的查询  
              5.图书馆  
                    图书馆藏书查询，个人借阅查询，书的续租，扫码查书
